# Terraform Implementation

## Status

🚧 Planned — Phase 2

The initial version of this project was deployed manually through the AWS Management Console to develop a practical understanding of the underlying AWS networking components.

The next phase will recreate the infrastructure using Terraform.

Planned Terraform-managed resources:

- Amazon VPC
- Public subnet
- Internet Gateway
- Route table and association
- Security Group
- EC2 VPN instance

The goal is to make the infrastructure reproducible using Infrastructure as Code.
